1. open your folder in VS Code.
2. open terminal (ctrl+shift+`).
3. npm init
4. entry point: server.js
5. author name: first_name last_name
6. npm install express
7. npm install dotenv (for environment variable)
8. npm install nodemon (for auto server restart)
9. create .env file at root directory and type PORT=8080 in it.
10. create .gitignore file at root directory (in vs code select a new file named .gitignore) and put the names: node_modules and .env inide this file at separate lines.
11. create a new file at root directory: server.js
12. setup express app and server by editing codes inside this file.
13. then setup routes
14. create public and views folder and create all html files in views folder.
15. all css, js, images, assets should be inside public folder.
16. open package.json file and under scripts, type: "start": "nodemon server.js" replacing the line with "test".
17. go to terminal and type: npm start
18. go to a browser and open url: "http://localhost:8080/" and you will see your web page on the screen.
19. after completing development of the project push it to your github.
20. then deploy the project to render and you will get a public url.
